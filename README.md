# k8s学习之路

> Kubernetes是云原生应用的基石

## 为什么学习k8s?

`Kubernetes`大势已定，从最开始的容器编排、调度、到集群管理，其在不同的阶段处理不同的问题，并且都解决得非常棒，发展到现在甚至有**云原生应用操作系统**之称。
